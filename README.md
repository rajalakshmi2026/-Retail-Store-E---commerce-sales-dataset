# E-Commerce Sales Data Cleaning Project

## 📌 About
This project demonstrates data cleaning and preprocessing of a messy e-commerce sales dataset using **Microsoft Excel** and **Power BI**.

## 📂 Files in this Repository
1. `dirty_ecommerce_sales_500x10.csv` - Raw dataset with data quality issues
2. `E Commerce Sales-Minni project.pdf` - Project documentation


## 🔧 Data Cleaning Process Using Excel & Power BI
- Removing duplicates using Excel's Remove Duplicates
- Handling missing values with Power Query
- Data type conversion in Power BI
- Date format standardization
- Text cleaning using TRIM, PROPER functions
- Outlier identification using Conditional Formatting

## 🛠️ Tools Used
- Microsoft Excel
- Power BI Desktop
- Power Query Editor

## 📊 Results
Cleaned dataset ready for dashboard creation and business insights in Power BI.

## 🚀 How to Use
1. Download `dirty_ecommerce_sales_500x10.csv`
2. Import to Excel or Power BI
3. Apply data cleaning steps
4. Build visualizations and dashboards
